# Kanyuk_Java

Решение тестового задания для кандидатов на стажировку QA Engineer (Java).

## Описание проекта
Программа написана на **Java 11** и объединяет в себе три подзадачи:
1. Интерактивный ввод двух целых чисел через консоль, их сравнение и выполнение базовых математических операций (сложение, вычитание, умножение, деление) с защитой от деления на ноль и некорректного ввода.
2. Интерактивный ввод двух строк и их посимвольное сравнение с выводом статуса идентичности.
3. Автоматический вывод всех чётных чисел из заданного массива от 1 до 10.
